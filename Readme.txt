Привет! Заранее спасибо за проверку моего проекта!
Проект был разделён на 3 под-проекта - библиотеки, консольного и оконного приложений.

Просьба лучше все запускать на операционной системе windows для избежания каких - либо непридвиденных обстоятельств.
Так же были использован Nuget-пакет NewtonSoft json, поэтому в случае "слёта" пакета просьба его подключить, или проверить из папки bin.

Логин и пароль в начале предоставляется самой программой. Зачем это нужно?
- Для того, чтобы распределить исполнителей и обычных юзеров.
Исполнитель умеет контролировать участниками и проектами
А обычный участник имеет доступ только к контролю проекта и ничего более.

Насчёт показа задач, я хотел максимально (и компактно) показать вам информацию про проекты, задачи, комментарии и записи.
Для того чтобы в проекте добавить задачу типа Epic , вам просто нужно зайти в Task Management => View => Sub-Task => Add sub-task => Epic 

Сделал для того, чтобы распределить высшие задачи , а также чтобы было легче видеть задачи высокого порядка.

Еще раз спасибо!
